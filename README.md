## Tips & Tricks

**Prompt weighting** ‚Äî Most models support parentheses to increase emphasis:
```
(beautiful eyes:1.3), (detailed face:1.2)
```

**Token order matters** ‚Äî Put the most important elements first. Models tend to weigh early tokens more heavily.

**CFG Scale** ‚Äî Lower CFG (5-7) = more creative/dreamy. Higher CFG (8-12) = more literal/sharp. Find your sweet spot.

**Sampling steps** ‚Äî 20-30 steps is usually enough for most samplers. More steps ‚â† always better.

**Seed locking** ‚Äî Found a composition you like? Lock the seed and change only a few words to iterate safely.

**VAE matters** ‚Äî For photorealistic results, make sure you're using a compatible VAE (e.g., `vae-ft-mse-840000`).
